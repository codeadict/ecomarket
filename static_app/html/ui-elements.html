Multiple category dropdown

Use hidden <select> tag with class "multiple-dropdown". This is fallback element, will also carry on user selection.
Print out .em-dropdown hidden element, it will replace select via JS.

Selectable items should have data-value="{category_id}" attributes, to connect choices with associated select.
When user selects something, associated hidden select will be updated with his choice.
